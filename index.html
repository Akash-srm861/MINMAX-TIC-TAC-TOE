How It Works — Minimax Algorithm (Unit 3)
State Space (Unit 1)
S₀ → Initial empty board
PLAYER(s) → Whose turn (X/O)
ACTIONS(s) → Empty cells
RESULT(s,a) → Board after move
TERMINAL-TEST(s) → Win/Draw?
UTILITY(s) → +10 / -10 / 0
Minimax Rule (Unit 3)
MAX player (X) → maximises score
MIN player (O) → minimises score
Leaf nodes → UTILITY value
Back-up from children
MAX node → picks max child
MIN node → picks min child
